# n8n + Submagic: Автоматизация видео-редактирования

Workflow на n8n для полной автоматизации редактирования видео в Submagic (субтитры, эффекты, монтаж).

## Проблема
Ручной монтаж коротких видео (Reels, Shorts) отнимал часы на проект. Нужно было масштабировать производство.

## Решение
- n8n цепочка: загрузка видео → отправка в Submagic API → применение шаблонов (субтитры, переходы, музыка) → возврат готового файла
- Интеграция с Telegram-ботом для запуска по запросу
- Оптимизация промптов для генерации субтитров и описаний

## Ключевые результаты
- Автоматизация 80–90% рутинного монтажа
- Сокращение времени на видео с 30–60 мин до <5 мин
- Масштабирование до десятков видео в день

## Стек
- n8n
- Submagic API
- Telegram Bot (опционально)
